Tailcast

Content management for public broadcast organisations, built on Wagtail, the popular open source CMS from Torchbox.

Modern

Built on Python, with a fast JavaScript user interface, Tailcast is designed for busy, digital-first newsrooms.

Seamless

Tailcast works with NPR and PBS APIs, making it easy for your editors to select and share content from across networks.

Open Source

Based on the latest version of Wagtail, the most popular Python CMS, used by Google, NASA and Mozilla.

Audio ready

Native support for audio content, with automatic conversion, transcription and tagging.

Headless

All your content is available through a well-documented API, so you can build the perfect front-end for your audience, or customise the included templates.

Mobile-ready

Tailcast includes a customisable native mobile app, ready for you to publish on the Apple and Google stores.

Cheap to run

Host it yourself on low-cost hardware, or use our managed cloud service.

Extensible

Use your Python, HTML and JavaScript skills to make Tailcast work exactly the way you want, or commission us to add new features.
